| --------- | ----------- | -------- | ---- |

| “ | Opening double quote | <kbd>⌥</kbd> <kbd>[</kbd> | &ldquo; |
| ” | Clos­ing double quote | <kbd>⌥</kbd> <kbd>⇧</kbd> <kbd>[</kbd> | &rdquo; |
